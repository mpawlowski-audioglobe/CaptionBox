# CaptionBox AV — Roadmap

## Najbliższy cel
- Stabilizacja napisów live bez zjadania końcówek zdań.
- Mniej powtórzeń i echa poprzednich hipotez ASR.
- Dopracowany ekran publiczności do zastosowań konferencyjnych.

## Planowane
- Zapisywanie ustawień.
- Profile AV: projektor, TV, LED wall.
- Diarization / rozpoznawanie mówców.
- Eksport transkrypcji.
